# AI Integration (Optional)

This folder is for optional AI prompt assets, configuration, and changelogs. If your project uses an LLM or external AI provider, add prompt templates and metadata here.

- Place prompt templates in `prompts/` (if needed)
- Track changes in `CHANGELOG_AI.md`
- See `/src/ai/` for runtime code and registry

If your project does not use AI, you can ignore or remove this folder.
